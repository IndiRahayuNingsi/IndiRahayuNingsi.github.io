About Me

Nama saya indi rahayu ningsi, nama panggilan indi. saya berasal dari mamasa sulawesi barat. saya besar dan tumbuh di kampung saya yang bernama rantebuda, dan dari SD-SMA saya bersekolah di kampung halaman saya. saya merupakan anak kedua dari pasangan yang bernama paulus dan ritawati. kakak saya bernama desti dan adik saya bernama greshia. saat ini saya sedang menempuh pendidikan di universitas negeri Manado fakultas teknik jurusan teknik informatika.
saya merupakan orang yang introvert. saya lebih suka menghabiskan waktu saya sendirian tanpa ada gangguan dari orang lain. hobi saya membaca buku, tidur, nonton drama korea dan streaming idola kpop. namun bukan berarti saya tidak suka belajar, saya suka belajar jika mata kuliah yang saya sukai seperti web dew, iu/ux, seni budaya.
saya tidak suka dengan orang yang berbasa-basi. ataupun orang yang hanya baik jika ada maunya. sehingga saya hanya mempunya sedikit teman dan itu membuat saya bahagia.
